Reason to use Embedded Styles

One key advantage of embedded styles is that they keep all your styling information in one central location within your HTML document. Unlike external stylesheets that require separate files, embedded styles contain all your CSS rules within the HTML file itself, making it easier to manage and maintain smaller websites.

Where Embedded Styles Are Placed on a web page

Embedded styles must be placed within the section of your HTML document. Here's an example showing both the placement and structure:

<!DOCTYPE html>
<html>
<head>
    <!-- Embedded styles go here -->
    <style>
        /* Your CSS rules go here */
        body {
            font-family: Arial, sans-serif;
        }
        
        h1 {
            color: blue;
        }
    </style>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>

Key Benefits

  • Keeps all styling in one central location
  • Ideal for single-page websites
  • Perfect for quick prototypes or demonstrations
  • Great for pages with unique styling that won't be reused

Reason to Use External Styles

External styles are used to define CSS rules in a separate file (usually ending in .css) rather than embedding them directly in the HTML file. This approach provides several advantages:

  • Reusability: One CSS file can be used across multiple HTML pages, making design changes faster and more consistent.
  • Separation of Concerns: Keeping structure (HTML) and styling (CSS) separate improves code organization and readability.
  • Reduced Page Size: By referencing a common CSS file, individual HTML files are smaller and load faster.

Where are external styles placed?

External style rules are written in a separate file, typically named something like styles.css. This file contains only CSS code and is saved in the project directory, often in a folder like /css or /assets/css.

How do web pages indicate they are using external styles?

Web pages include external stylesheets by placing a <link> tag inside the <head> section of the HTML document. For example:

<link rel="stylesheet" href="styles.css">

This tells the browser to fetch and apply the styles from the specified CSS file.
